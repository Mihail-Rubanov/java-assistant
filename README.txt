Основные команды для работы в git:

git init - инициализация репозитория;

git add <имя.расширение> - подготовить <файл> к коммиту;

git commit -m "Комментарий к коммиту." - сделать коммит и оставить комментарий;

git log - вывести подробную историю коммитов;

git status - показать текущее состояние репозитория;

pwd - показать, в какой ты папке;

ls - показать файлы и папки в текущей папке;

cd - перейти в другую папку;

touch - создать файл;

cat - распечатать содержимое текстового файла.
